Assets.Map Namespace
Contains the logic for the construction and maintenance of the map. This includes MapGraph, MapNode, and MapEdge, along with some graph traversal algorithms.
Classes
  ClassDescription
EdgeOrientationUtil
A simple utility class that's used to help align map edges with their endpoints.
MapEdge
The standard implementation of MapEdgeBase.
MapEdgeBase
The abstract base class from which all map edges derive. A map edge is a connection between two map nodes upon which highways can be built. The MapEdgeBase class is provided primarily for convenience, and a more performant Map paradigm would likely not include them in their current form.
MapEdgeEventArgs
An EventArgs class used during events involving map edges.
MapGraph
The standard implementation for MapGraphBase, which represents the game map as an undirected graph. Acts as the factory, adjacency canon, and pathfinder for all nodes and edges.
MapGraphAlgorithmSet
The standard implementation of MapGraphAlgorithmSetBase, which uses Djikstra's and BFS to compute its values.
MapGraphAlgorithmSetBase
An abstract base class that defines a set of algorithms MapGraph needs in order to function correctly.
MapGraphBase
An abstract base class that manages the in-game map. Maps are represented as undirected graphs. Acts as the factory, adjacency canon, and pathfinder for all nodes and edges.
MapGraphException
A general exception for all MapGraph-related errors that aren't easily covered under one of the standard exceptions.
MapNode
The standard implementation of MapNodeBase. Maps in the game are undirected graphs, with highways situated along edges and all other features situated on nodes.
MapNodeBase
The abstract base class that defines a map node. Maps in the game are undirected graphs, with highways situated along edges and all other features situated on nodes.
MapNodeEventArgs
A simple EventArgs class that returns Nodes on events that trigger in relation to them.
MapNodeException
An exception class for errors related to MapNodes that don't fall easily into more standard categories.
MapNodeUISummary
A class containing information that MapNode should pass into UIControl whenever it catches user input.
Neighborhood
A class used to facilitate the use of prefabs in map creation at design time.
NodeDistanceSummary
Used to return results on some MapGraphBase and MapGraphAlgorithmSetBase methods.
TerrainGrid
The standard implementation of TerrainGridBase. This is used in tandem with MapGraphBase to define and display a cohesive map.
TerrainGridBase
Abstract base class for handling the generation of visible terrain in the form of a hex grid. This is used in tandem with MapGraphBase to define and display a cohesive map.
TerrainHexTile
A hexagonal tile used by TerrainGrid to visualize the map defined in MapGraphBase.
TerrainMaterialRegistry
A simple class that manages the materials for each terrain type.
