MapGraphAlgorithmSetGetNearestNodeToEdgeWhere Method

Gets the nearest node to the endpoints of a given edge of origin that satisfies a specified condition. Will only return nodes within a specified maximum distance, and reports the shortest distance from one of the two endpoints without specifying which endpoint is closer.
