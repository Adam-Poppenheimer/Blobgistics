Assets.ConstructionZones Namespace
Contains the logic that handles ConstructionZones and the ConstructionProjects that control them. Elements in this namespace define the ways in which players can construct Societies, ResourceDepots, HighwayManagers, and clear and plant forests.
Classes
  ClassDescription
ClearForestConstructionProject
Defines and executes the clearing of forests.
ConstructionProjectBase
The abstract base class for all construction projects, which do the main work of determining what gets built, what it costs, and where it can go.
ConstructionProjectUISummary
A class containing information that the UI might need to know about a given construction project.
ConstructionZone
The standard implementation for ConstructionZoneBase. It enable the creation of societies, resource depots, highway managers, and the addition and removal of forests by the player.
ConstructionZoneBase
The abstract base class for all construction zones, which enable the creation of societies, resource depots, highway managers, and the addition and removal of forests by the player.
ConstructionZoneException
An exception for communicating errors involving construction zones that aren't covered by the standard exceptions.
ConstructionZoneFactory
The standard implementation for ConstructionZoneFactoryBase. Handles creation and destruction of construction zones, keeps a record of available construction projects, and maintains construction zone placement restrictions.
ConstructionZoneFactoryBase
The abstract base class for all construction zone factories. Handles creation and destruction of construction zones, keeps a record of available construction projects, and maintains construction zone placement restrictions.
ConstructionZoneUISummary
A class containing information that ConstructionZoneBase should pass into UIControl whenever it catches user input.
FlexibleCostConstructionProjectBase
The abstract base class for construction projects who have flexible costs.
HighwayManagerConstructionProject
Defines and executs the construction of a highway manager.
PlantForestConstructionProject
Defines and executes the planting of forests on grassland.
ResourceDepotConstructionProject
Defines and executes the construction of a resource depot.
TierOneSocietyConstructionProject
Defines and executes the construction of a particular tier 1 society.
