UnityCustomUtilities.Grids Namespace
Contains classes for a hexagonal grid of arbitrary tiles. This implementation is based off a detailed blog from Red Blob Games (http://www.redblobgames.com/grids/hexagons/), which was also source of much of the code here, though it's been modified since then.
Classes
  ClassDescription
FractionalHexCoords
Represents an alternate, fractional hex coordinate used mostly for line drawing and determining the nearest hex coordinates of a given point in space.
HexCoords
Represents a coordinate of a hex tile in cubic coordinates, and contains a number of methods for interacting with such coordinates.
HexGridT
The standard implementation of HexGridBase, which defines a grid of hexagonal tiles.
HexGridBaseT
An abstract interface defining a grid of hexagonal tiles that can be queried in a number of ways.
HexGridOrientation
Contains information necessary to define the orientation of hexes in a grid.
HexTileBase
The abstract base class for a tile in a hexagonal grid.
HexTilePathingLogicBaseT
The abstract base class for the pathing logic of a hex grid. Determines the cost of entering hex tiles and what hex tiles are completely impassable.
