Remarks
Note that when the number of blobs in blobsToAlign is greater than BlobsPerRow * BlobsPerColumn, all remaining blobs will not be moved. This is a problem with the alignment strategy that was of so little consequence to the end user experience that its resolution was never a priority. It remains a valid target for refactoring, however.
