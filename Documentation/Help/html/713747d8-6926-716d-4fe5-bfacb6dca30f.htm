MapGraphAlgorithmSetBaseGetShortestPathBetweenNodes Method

Gets the shortest path between the starting and ending nodes, traversing only the nodes specified in allNodes. If a path exists, returns a list whose first and last elements are the starting and ending points, with a number of intervening nodes that represent a valid path. Otherwise, the list is just the starting node.
