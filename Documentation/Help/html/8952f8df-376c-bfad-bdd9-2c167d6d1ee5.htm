Assets.Session Namespace
Contains all the logic associated with session, serializiable records of the state of a given map. Maps and saved games are stored as sessions, which are exchanged between persistent storage and the current runtime via the SessionManager class. This namespace also contains interaction with the file system through FileSystemLiasion.
