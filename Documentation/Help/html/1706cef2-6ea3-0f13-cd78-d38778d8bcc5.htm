Assets.BlobSites Namespace
Contains the logic for storing and placing blobs in various locations. Every MapNode contains a BlobSite that manages its contents, placement permissions, extraction permissions, and capacities.
Classes
  ClassDescription
BlobAlignmentStrategyBase
The abstract base class for all blob alignment strategies, which manage the positioning of resource blobs contained within a BlobSiteBase.
BlobSite
An entity capable of having resource blobs placed into or extracted from it, and having its placement permissions, extraction permissions, and capacity changed.
BlobSiteBase
The abstract base class for all blob sites, which are were all resource blobs are created, and also where they're stored when not traveling through highways.
BlobSiteConfiguration
The standard implementation for BlobSiteConfigurationBase.
BlobSiteConfigurationBase
The abstract base class for all blob site configurations, which instruct blob sites where to destroy their blobs and how to organize their contents in space.
BlobSiteException
The exception class for errors related to blob sites that don't fit under the standard exceptions.
BlobSitePermissionProfile
A POD class that stores BlobSite placement permission, extraction permission, per resource capacity, and total capacity in a more portable form.
BlobSiteUISummary
A class containing information that BlobSite should pass into UIControl whenever it catches user input.
BoxyBlobAlignmentStrategy
An alignment strategy that arranges blobs in a box of a certain width and height, with a certain number of blobs per row and column within the box.
