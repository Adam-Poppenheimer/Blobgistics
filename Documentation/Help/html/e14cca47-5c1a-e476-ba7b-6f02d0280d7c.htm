MapNodeBase Properties

The MapNodeBase type exposes the following members.

Properties
  NameDescription
AssociatedTiles
The tiles in the TerrainGridBase that are associated with this MapNodeBase. Association is determined by distance.
BlobSite
The BlobSite that defines the resource contents of this node.
ID
An ID number unique across all other map nodes. This value is not guaranteed to be a unique key against other object types.
Neighbors
The neighbors directly connected to this node by a single edge.
ParentGraph
The MapGraphBase that's supposed to be managing this edge.
Terrain
The terrain of the node, which affects appearance, society complexification, and construction project placement.
TerrainGrid
The TerrainGridBase that the MapNodeBase should use to define its region in the world.
TerrainMaterialRegistry
The TerrainMaterialRegistry that the MapNodeBase should use to inform its appearance.
UIControl
The UIControlBase that this node should send input events (like PointerDown) to.
