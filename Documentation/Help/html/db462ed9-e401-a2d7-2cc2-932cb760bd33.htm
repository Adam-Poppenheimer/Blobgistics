Contains UI elements associated with construction zones and construction projects.
Classes

The standard implementation of ConstructionPanelBase. This gives the player the ability to select various construction projects and build construction zones on them.

The abstract base class for construction panels, which give the player the ability to select various construction projects and build construction zones on them.

The standard implementation of ConstructionZoneSummaryDisplayBase, which provides information and commands for a particular construction zone.

The abstract base class for construction zone display, which provides information and commands for a particular construction zone.
