Remarks
Since Unity doesn't serialize Dictionaries but I needed to establish several mappings between ResourceType and other values, I decided to use the equivalence of enums and ints to fudge a dictionary-like object into existence. This implementation is not robust to modifications on ResourceType. Changing the ResourceType enum will likely break any instance of this class or any of its subclasses.
